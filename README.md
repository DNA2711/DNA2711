Software Engineering
--------------------

I have been immersed in coding for three years now. Before stepping into the realm of programming, my true passion lay in computer hardware. I was deeply fascinated by technological devices and computer components. However, one day, I came to the realization that it would be advantageous for me to delve into software development while maintaining a solid understanding of hardware. This perspective is akin to being a full-stack engineer, but with a distinctive focus on the software and hardware aspect. As a technologist, I find the synergy between software and hardware to be a compelling and holistic approach. My journey reflects a conscious choice to not only appreciate the intricacies of coding but also to comprehend the inner workings of the physical components that constitute a computing system. This combination of skills positions me as a versatile IT professional, capable of navigating both the software and hardware dimensions of technology.
